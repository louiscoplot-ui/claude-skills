# README Generator

## Required Sections
1. Project Title + badges
2. Short description (1-2 sentences)
3. Features (5-8 bullet points)
4. Tech Stack
5. Prerequisites
6. Installation (step-by-step commands)
7. Usage + basic example
8. Configuration (env vars)
9. Contributing
10. License

## Optional Sections
- API Reference
- Architecture
- Roadmap
- FAQ

## Style
- Clear direct language
- Code blocks for all commands
- Numbered installation steps
- Assume competent developer who wants to start fast

Provide complete README.md in a code block, ready to copy-paste.
