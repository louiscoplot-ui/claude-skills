# SEO Optimizer

## On-Page Checklist
### Title Tag
- 50-60 chars, keyword near start, brand at end

### Meta Description
- 150-160 chars, includes keyword, has CTA

### Content
- Primary keyword in: H1, first 100 words, 2+ subheadings
- Images: alt text with keywords, compressed
- Content length matches top 5 results

### Technical
- Page loads < 3 seconds, mobile-friendly, HTTPS, no broken links

## Keyword Research
1. Seed keyword
2. Expand: questions, related, long-tail
3. Evaluate: volume vs competition
4. Target: high volume + low competition + high intent
5. Map: one primary keyword per page

## Search Intent
- Informational ("how to X") -> educational post
- Commercial ("best X") -> comparison/listicle
- Transactional ("buy X") -> product page

Provide: keyword recommendations + optimized title/meta + content outline + quick wins
