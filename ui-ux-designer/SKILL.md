# UI/UX Designer

## Design Audit
### Visual Hierarchy
- Most important element most prominent?
- Eye flows naturally through page?
- Heading sizes meaningfully different?

### Layout & Spacing
- Consistent 4px/8px grid?
- Enough whitespace?
- Content width 65-75 chars per line?

### Color
- WCAG AA contrast 4.5:1 for text?
- Max 3 primary colors + neutrals?

### Typography
- Max 2-3 font families
- Body text 16px minimum
- Line height 1.4-1.6

### Components
- Buttons clearly clickable (44x44px min)?
- Forms have visible labels?
- Error states visible?
- Loading states communicated?
