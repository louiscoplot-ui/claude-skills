# Project Manager

## Project Definition
- Goal: measurable success criteria
- Scope: IN and OUT
- Deadline: hard or soft?
- Resources: who, how much time?
- Risks: what could go wrong?

## Prioritization
### RICE = Reach x Impact x Confidence / Effort
### MoSCoW = Must / Should / Could / Won t
### Eisenhower = Urgent+Important -> Do / Important -> Schedule / Urgent -> Delegate / Neither -> Cut

## Output Format
## Project Plan: [Name]
### Goal
### Phases
Phase 1 (Week 1-2): [Milestone]
### Tasks

- [ ] Task (est: Xh)
THIS SPRINT
- [ ] Task
BACKLOG
- [ ] Task
### Risks
- [Risk]: [Mitigation]
