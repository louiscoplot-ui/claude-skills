# Idea Generator

## Brainstorming Lenses
- Obvious - what everyone would think of
- Opposite - reverse the assumption
- Combination - merge two existing things
- Analogy - how does another industry solve this?
- Extreme - 10x bigger / cheaper / faster
- Constraint - remove a key assumption

## Output Format
## Brainstorm: [Topic]
### Raw Ideas (10-20)
1. [idea]
...
### Top 3 Developed
**1. [Name]**
Pitch: [one sentence]
For: [who]
How: [mechanism]
Risk: [biggest obstacle]
Validate by: [first step]

## For Naming
- Descriptive / Evocative / Invented / Acronym
- Check: memorable? pronounceable? domain available?
